## FixBlox SEO Expansion Roadmap (Trade + Local SEO)

This roadmap outlines the next SEO growth steps for FixBlox, covering trade landing pages, city/region landing pages, demo trade websites, and schema enhancements.

---

## 1. Trade Landing Page Expansion (Phase 1)

Create additional trade-specific landing pages to broaden organic reach.

### Priority Trades

- **Plumbers** (DONE)
- **Electricians** (DONE)
- **Roofers**
- **Builders**
- **Painters & Decorators**
- **Gardeners / Landscapers**
- **Handyman Services**
- **Carpenters / Joiners**
- **HVAC / Heating Engineers**
- **Cleaners & Domestic Services**

### Tasks

- **Add metadata for each trade**
  - Unique title + meta description per trade.
  - Trade-specific keywords (e.g. “plumber websites”, “electrician website builder”).

- **Add unique benefits list per trade**
  - Bullet-point benefits tailored to how each trade works (emergency work, quoting complexity, seasonal demand, etc.).

- **Add tailored CTA blocks**
  - “Get Your [Trade] Website Quote” / “See [Trade] Demo Site”.
  - Include both QuoteFlow and Full Website Builder CTAs where relevant.

- **Add trade-specific images**
  - Hero visuals per trade (e.g. plumbing van, electrician at work).
  - Alt text with trade + service context.

- **Add internal links between trade pages**
  - “Also great for…” section linking to related trades.
  - Ensure each trade page links back to:
    - Homepage
    - QuoteFlow
    - Full Website Builder
    - Contact

## 4. Schema Expansion (Phase 4)

Improve site authority and SERP visibility using structured data.

### Add to Each Page

- **Website schema**
  - Already added at the global level; ensure new pages still fit this model.

- **Organization schema**
  - Global org schema already present; ensure any new properties (social profiles, contact options) are kept in sync.

- **Breadcrumb schema**
  - Already in place for blog posts and industry pages.
  - Add for:
    - Demo sites (`Home > Demo > [Trade]`).
    - City landing pages (`Home > Cities > [Trade in City]`).

### Add to Trade / City Pages

- **Service schema**
  - `@type: Service` describing “Trades Website Builder”, “QuoteFlow for [Trade] in [City]”, etc.

- **LocalBusiness schema**
  - Each city page should describe **a local trade website provider** with areaServed = target city/region.

- **FAQ schema (if relevant)**
  - For pages with FAQ sections (pricing, timelines, what’s included), add FAQPage JSON-LD.
